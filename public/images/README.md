# Workshop Images

This folder is for storing your workshop photos.

## Recommended Images

- Hero image: Main workshop or tractor service photo (1200x800px recommended)
- About section: Workshop interior or team photo
- Gallery images: 
  - Tyre repair bay
  - Tractor maintenance
  - Workshop equipment
  - Mobile service vehicle

## How to Add Your Photos

1. Add your image files to this folder (e.g., `hero.jpg`, `workshop-1.jpg`)
2. Update the image `src` paths in `src/App.js` to use your local images:
   - Change from: `src="https://images.unsplash.com/..."`
   - Change to: `src="/images/hero.jpg"`

## Image Formats

- Supported: JPG, PNG, WebP
- Recommended size: 1200px width for hero images, 800px for gallery images
- Optimize images before uploading for faster page loads
